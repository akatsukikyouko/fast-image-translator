# fast-image-translator

免费快速的网页图片翻译的油猴（Tampermonkey）脚本，得益于 Qwen3.5 这样强大的可在本地部署的多模态模型的能力，快速翻译漫画、新闻等外语图片为简体中文！就连算力都在本地，也就是说，一切都是无限制+免费的！！！

## 功能介绍

- **智能图片翻译**：基于多模态大语言模型，自动识别图片中的文字并翻译为简体中文
- **归一化坐标定位**：使用 0-1000 坐标系统精确定位文本区域
- **智能字号识别**：自动判断文字大小（small/medium/large）
- **文本智能合并**：同一行或相邻的文本片段会合并为一个整体，避免过度拆分
- **多种翻译模式**：
  - 自动模式：智能选择图片替换或文字弹窗
  - 文字翻译模式：适用于无法直接替换图片的网站（如 X/Twitter）
  - 一律文字翻译：可针对特定网站强制使用文字翻译模式
- **自定义配置**：支持配置 API 接口、模型名称、密钥等
- **额外参数支持**：支持添加自定义 JSON 参数到请求体中
- **JSON 修复引擎**：内置强大的 JSON 修复功能，自动处理模型输出的格式问题

## 模型测试结果

### ✅ 正常推荐

| 模型 | 类型 | 备注 |
|------|------|------|
| qwen3.5-35b-a3b | AWQ 非思考 | 本地部署推荐 |
| qwen3.5-27b | Q4 量化 | 本地部署推荐 |
| qwen3.5-flash | API | 在线 API 推荐 |
| glm-4.6v-flash | API | 在线 API 推荐 |
| kimi-2.5 | API | 杀鸡焉用牛刀 |

### ⚠️ 可用

对于这两个模型，真要用还是建议使用Q6以上的量化。

| 模型 | 类型 | 备注 |
|------|------|------|
| qwen3.5-9b | Q4 量化 | 基本可用 |
| glm-4.6v-flash | Q4 量化 | 基本可用 |

### ❌ 不推荐

- **qwen3.5-4b（q4） 以下模型**：输出json格式不稳定，基本不可用
